# BLU-PATH: Embedded Web-Based Indoor Navigation System_ECE_KPRIET

## Overview

BLU-PATH is an embedded web-based indoor navigation system developed using the Silicon Labs BRD2605A (SiWx917) development board. The system provides campus navigation through a locally hosted Wi-Fi network and embedded web server, allowing users to access navigation information without internet connectivity or a mobile application.

## Key Features

* Local Wi-Fi Access Point
* Embedded Web Server
* QR Code Based Access
* Campus Navigation Portal
* Internet-Free Operation
* Lightweight and Cost-Effective Solution

## Hardware Platform

* Silicon Labs BRD2605A (SiWx917)

## Technologies Used

* Embedded C
* Wi-Fi Networking
* HTTP Web Server
* HTML
* CSS
* JavaScript
* Simplicity Studio
* WiseConnect SDK
